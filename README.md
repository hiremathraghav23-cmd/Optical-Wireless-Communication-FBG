# Optical Wireless Communication Using Fiber Bragg Grating

## Project Overview
This project focuses on improving optical signal transmission efficiency in Optical Wireless Communication (OWC) systems.

Optical Wireless Communication uses light waves instead of traditional electrical signals for data transmission. The project aims to enhance signal quality and transmission performance using Fiber Bragg Grating (FBG) technology.

## Objectives
- Improve optical signal transmission efficiency
- Analyze signal performance in wireless optical communication
- Study the effect of Fiber Bragg Grating in signal filtering

## Tools and Technologies Used
- MATLAB
- OptiSystem
- Optical Communication Concepts

## Methodology
The Optical Wireless Communication system was designed and simulated using MATLAB and OptiSystem. Fiber Bragg Grating was used to enhance signal stability and reduce noise. Various system parameters were analyzed to evaluate communication performance.

## Results
The simulation results showed improvement in signal quality and transmission efficiency when Fiber Bragg Grating technology was used in the system.

## Conclusion
This project demonstrates how Fiber Bragg Grating can improve the performance of Optical Wireless Communication systems and enhance signal reliability.
